## 如何开始

1. **安装插件**：在微信中搜索并添加 ChatGPT on WeChat 插件。
   ```
   #installp git@github.com:Sakura7301/game.git
   ```

2. **配置插件**：请务必在 ChatGPT on WeChat 的配置文件中设置 `hot_reload = true`，以确保插件能够正常运行。

3. **加载游戏配置**: 加载用于游戏系统的配置文件。
配置文件的模板在插件根目录的`config-template.json`中，需复制该模板创建最终生效的 `config.json` 文件：
   ```bash
   cp config-template.json config.json
   ```

4. **认证管理员**: 参见游戏指令说明文档，认证管理员。

5. **开始游戏**：开始游戏(插件开启后默认为开机转台，如若 `关机`，则需要重新执行 `开机` 指令之后才能够正常交互)。
